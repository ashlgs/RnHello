## 前言

　　`xueyuan-cms-app`

### 项目介绍

　　xueyuan-cms-app`
    
####    工程目录：
#####   pages:  各个页面组件
            四个主页面：
            Home.js：首页
            Video.js：视频
            SmallVideo.js：小视频
            QA.js：问答
#####     components: 通用组件
#####    constant:   常量表
#####  styles: 公共样式
        cokors.js:颜色表
        dimension.js:通用组件宽高、页面边距等
        font.js:字体字号设置
       
#####     utils:  通用工具类
          RequestUtil.js:网络请求工具
          Utils.js:通用方法
#####     assets: 资源文件夹，放置图片等
    
 #####     RouteConfig.js:   路由配置，每个页面都要在这里配置
 #####   TabNavigator.js:    首页底部导航配置
